EMDR Light Bar Mage is an advanced web application for EMDR (Eye Movement Desensitization and Reprocessing) bilateral stimulation. It combines a visual light bar, synthesised tones, haptic/midi/serial extensions, session recording, and real‑time synchronisation over WebSocket. The interface is designed to be fully customisable, supporting therapeutic bilateral stimulation in a clinical or home‑practice setting. For full documentation, please see: https://github.com/robynsmith248-alt/Light-Bar-Mage/blob/main/readme.pdf
